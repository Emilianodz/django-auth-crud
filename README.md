# django-auth-crud

A Django web application that allows users to perform CRUD operations on a database and verify your users.

### Requirements 

* Python 3.10.1
* asgiref==3.6.0
* Brotli==1.0.9
* dj-database-url==1.2.0
* Django==4.1.5
* gunicorn==20.1.0
* psycopg2-binary==2.9.5
* sqlparse==0.4.3
* tzdata==2022.7
* unicorn==2.0.1.post1
* whitenoise==6.3.0
